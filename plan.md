# 斗地主游戏开发计划

## 项目概述
- **项目名称**: 欢乐斗地主 (Dou Dizhu)
- **项目类型**: Python桌面游戏
- **核心功能**: 三人斗地主扑克游戏，支持人机对战
- **目标用户**: 休闲游戏玩家

## 技术选型
- **开发语言**: Python 3.8+
- **GUI框架**: Pygame (适合2D游戏开发，灵活度高)
- **打包工具**: PyInstaller (支持多平台构建)

## UI/UX 设计方向

### 整体视觉风格
- 中国传统风格，融合现代简洁设计
- 绿色为主色调（经典斗地主风格）
- 精致的卡片设计和动画效果

### 色彩方案
- 主色: #2E7D32 (森林绿)
- 背景: #1B5E20 (深绿)
- 强调色: #FFD700 (金色)
- 文字: #FFFFFF (白色)

### 布局设计
- 屏幕尺寸: 1280x720 (16:9)
- 玩家位置: 上方(电脑B)、下方(玩家)、右侧(电脑A)
- 底牌和地主牌显示在屏幕中央
